## Аннотация

В рамках курсового проекта разработана система контроля температуры на базе микроконтроллера Arduino Uno. Устройство обеспечивает измерение температуры окружающей среды с помощью цифрового датчика DHT11, отображение текущих показаний на LCD-дисплее 1602 (I2C) и светодиодную индикацию при превышении заданного порогового значения (25°C). Система функционирует в автономном режиме с периодичностью опроса датчика 2 секунды.

## Схема подключения

| Компонент | Контакт Arduino | Назначение |
|-----------|-----------------|------------|
| DHT11 | D2 | Сигнальный вывод |
| Светодиод (+) | D3 | Анод |
| Светодиод (-) | GND | Катод через резистор 220 Ом |
| LCD I2C (SDA) | A4 | Передача данных |
| LCD I2C (SCL) | A5 | Синхронизация |

Принципиальная схема представлена в папке `screenshots/`.

## Программное обеспечение

**Среда разработки:** Arduino IDE 2.x  
**Язык программирования:** C++ (Arduino Framework)  
**Используемые библиотеки:**
- `LiquidCrystal_I2C.h` — управление дисплеем по шине I2C
- `DHT.h` — работа с датчиком серии DHT

**Алгоритм работы:**
1. Инициализация периферийных устройств (датчик, дисплей, порт светодиода)
2. Опрос датчика с интервалом 2000 мс
3. Проверка корректности полученных данных
4. Вывод текущего значения температуры на экран
5. Сравнение температуры с пороговым значением 25°C
6. Управление состоянием светодиода в зависимости от результата сравнения
7. Вывод статуса системы на дисплей

Исходный код программы размещён в папке `code/`.

## Результаты тестирования

| Условие | Ожидаемая реакция | Фактический результат |
|---------|-------------------|----------------------|
| Температура ≤ 25°C | Светодиод выключен, индикация «HEATING OFF» | Соответствует |
| Температура > 25°C | Светодиод включён, индикация «HEATING ON» | Соответствует |
| Отсутствие сигнала с датчика | Вывод сообщения «Sensor Error!» | Соответствует |

Разработанная система функционирует устойчиво. Отклонений от заданных параметров не выявлено.
